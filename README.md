# Mini-Shopping-Mall

## 프로젝트 개요

간단한 필터 기능을 갖춘 쇼핑몰을 구현하는
' _유튜브 드림코딩_ ' 엘리님의 강의를 클론코딩한 프로젝트입니다.



## 주요 기능

1. 아이템 타입별(tshirts, pants, skirt), 컬러별(pink, yellow, blue) `필터링` 기능
2. 아이템 `스크롤링` 기능



## 사용 스택



- HTML
- CSS
- JAVASCRIPT

## 프로젝트 기간
